QuickEdit makes any pages editable locally by adding the contentEditable attribute to the HTML. Any changes are not permanent and will be reverted when the page is reloaded. QuickEdit is meant for developers and designers who need to perform occasional test edits on the front end without any commitments. QuickEdit can be accessed via shortcut keys **(ALT+R)** or through the context menu.

The shortcut key is also configurable via the chrome extensions shortcut menu.
